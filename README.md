# AI Chat Webapp

精美 AI 对话网页，支持接入 OpenAI 兼容 API。

## 功能

- 🎨 精美深色主题，毛玻璃效果
- 💬 多对话管理（新建/切换/删除）
- 🔌 真实的 OpenAI 兼容 API 接入
- ⚡ SSE 流式输出，打字机效果
- 📝 Markdown 渲染 + 代码高亮 + 一键复制
- 🧠 可配置模型、温度、TopP、MaxTokens、系统提示词
- 📱 完全响应式，移动端适配
- 💾 本地持久化存储

## 使用

1. 打开页面
2. 点击左下角「API 设置」
3. 填写 API 地址、Key、模型名称
4. 开始对话！

## 部署

纯静态页面，直接部署到 GitHub Pages 或任意静态托管服务。
